在VMware中安装Kali Linux

发表于 2021-04-15 更新于 2021-04-22 分类于 学习心得 阅读次数： Valine：

本篇文章记录了笔者如何使用 VMware 安装 Kali Linux 虚拟机。

什么是 Kali？

Kali Linux 是基于 Debian 的 Linux 发行版，设计用于数字鉴识和渗透测试。由 Offensive Security Ltd 维护和资助。 Kali Linux 既有 32 位又有 64 位的映像。可用于 x86 指令集的中央处理器。维基百科

下载

进入 Kali Linux 官网，根据自己的系统选择合适的版本进行下载。

笔者选择的为此时最新发布的版本（kali-linux-2021.1-installer-amd64.iso）。

安装

安装 Hexo 只需十几分钟时间，若在安装过程中遇到问题或无法找到解决方式，请善于谷歌。

安装前提

安装 Kali 相当简单，只需要先安装下列应用程序即可：

  • VMware

笔者的电脑已经预先安装了 VMware Workstation Pro 16，本文不再展示如何安装。

安装 Kali

安装前的虚拟机设置

1.打开 VMware，选择创建新的虚拟机

2.默认选择典型配置即可，如有需求，则选择自定义。

这里笔者没有其他需求，选择为典型。

以下为选择典型后的操作，自定义同理。

3.浏览找到刚才下载的光盘映像文件（后缀为.iso）并选择。

4.客户端操作系统选择 Linux

版本选择 Debian，这里笔者选择了 Debian 10.X 64 位

5.设置虚拟机名称和安装位置

6.分配最大磁盘大小，这里建议>20GB，笔者选择了 30GB

虚拟磁盘选择存储为单个文件或拆分为多个文件均可，这里笔者选择了多个

7.上述步骤执行完毕后，单击完成后会生成一个新的虚拟机，点击开启此虚拟机。

进入安装界面

8.选择 Graohical install(图形的安装)

9.语言选择中文，地区选中国，键盘选汉语。

10.设置主机名，域名留空，设置用户名和密码。

磁盘分区

11.这里初次安装时，笔者遇到了一些问题，故照抄了 csdn 上某大佬的操作，这里附上原地址，根据图示选择即可。

“这里我建议选择手动吧　继续　选择刚刚在 Vmware 中新建的硬盘　继续　这是一定要选择“是”　继续，后面的如果是新手或者不确定的根据图中我的选择来吧”

安装基本软件

12.笔者这里没遇到坑，如果是按照笔者的步骤来的话，到这一步直接点继续，耐心等待安装完成即可。

安装 GRUB 引导

选择是和/dev/sda，确定。

安装完成

安装完成后会自动重启进入系统，接着输入刚才设置的用户名和密码，就可以顺利登入系统。

最后进入桌面，如果没错的话桌面大概如下图所示（版本不同可能会有出入），这样就算大功告成了，之后就可以愉快的享受 Kali Linux 了！

写在最后

这篇文章写得比较仓促，所以相关的知识也比较简略，大概算是一个用于科普的随手记，主要是写给自己看的，以防自己忘记。

本以为装一个 Kali 虚拟机非常简单，因为笔者曾经用 VMware 装过 CentOS8 和 Win10，当初安装这个系统时，VMware 一下就识别出来了，而且还是几乎全自动式的安装，非常傻瓜，导致分区这块知识不太了解，换了 Kali 以后立即原形毕露了，果然我还是 too young too simple sometimes naive ，今后还是要多多虚心学习有关于系统分区以及 Linux 的相关知识呀~
